# Security Policy

## Supported Versions

:white_check_mark: = Full support | 
☑️ = Partial Support |
:x: = No Support


| Version | Supported          |
| ------- | ------------------ |
| bwe 1.x.x   | :white_check_mark: |
| old licensed builds of bwe   | :x:                |
| bwr <=1.12.1   | ☑️                |
| bwr >=1.12.1   | :x:                |
